# Project Overview
This project focuses on predicting building energy consumption using machine learning techniques, based on architectural and building-related features. 
The goal is to demonstrate how data science can support energy-efficient building analysis and decision-making.
The dataset is sourced from NYC Building Energy Benchmarking data, which contains real-world information about building type, size, age, and energy performance.

# Dataset link
https://www.kaggle.com/datasets/mwaterman/manhattan-nyc-building-energy-data-2022

# Conclusion
The model successfully learns meaningful patterns from real-world building data.
Architectural features significantly influence energy consumption.
While not perfect, the model provides interpretable and practical insights.
This project demonstrates how machine learning can support sustainable building analysis.
